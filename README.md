# dataset-DZ6 Набор данных по туризму Новой Зеландии
https://www.kaggle.com/datasets/digitalashish/nz-tourism-dataset-2024
Этот набор данных содержит исторические о количестве прибывающих в Новую Зеландию туристов, Он помогает анализировать тенденции в сфере туризма, модели восстановления и планировать будущее для туристической отрасли. Прогнозы основаны на эконометрическом моделировании, текущих тенденциях и наиболее достоверных прогнозах международных факторов, а также разработаны с учетом мнения представителей туристической отрасли.

Статистика туризма предоставляет информацию о туристической индустрии Новой Зеландии, в том числе:

Расходы внутренних и иностранных туристов
Продолжительность пребывания иностранных студентов в стране менее 12 месяцев.
Вклад туризма в ВВП
занятость в сфере туризма
Прибытие и отъезд иностранных туристов и жителей Новой Зеландии

ГИПОТЕЗЫ:

Эластичность по доходу, существует прямая корреляция между ростом ВВП в странах-донорах (Китай, США) и количеством прибытий из этих стран.

В периоды нестабильности в Азиатско-Тихоокеанском регионе, Новая Зеландия как безопасная гавань, получает прирост туристов.

МОДЕЛИ прогнозирования ARIMA, SARIMA, ETS, PROPHET и XGBOOST.
